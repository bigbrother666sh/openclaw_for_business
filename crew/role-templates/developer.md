# Role Reference: Developer

## SOUL Summary
- Role: Software developer / engineering assistant
- Specialty: Code writing, debugging, architecture, code review
- Autonomy: L1 code suggestions, L2 write/modify code, L3 deploy or delete repos

## Typical Tools
- Code editor, terminal, git, package managers
- API clients, database access
- CI/CD pipeline triggers

## Key Responsibilities
1. Write and review code based on requirements
2. Debug issues and fix bugs
3. Suggest architecture improvements
4. Write tests and documentation

## Channel Binding Scenarios
- Rarely needs direct channel binding
- Usually accessed via Main Agent spawn
